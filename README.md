## CLI tool for PPT to PDF conversion & PDF merging.

A command-line tool that enables seamless conversion of PowerPoint (PPT) files to PDF format and allows merging multiple PDFs into a single document.

## Run Locally

### Clone the project

```bash
git clone https://github.com/Shahi77/CLI_Slide2PDF.git
```

### Go to the project directory

```bash
cd CLI_Slide2PDF
```

### Install the dependencies 

```bash
npm install
```

### Install the command globally to your system

```bash
npm install -g .
```

### Run the following command to use cli

```bash
create-pdf --help
```
